# Customer Segmentation and Market Basket Analysis

## Project Overview
This project analyzes online retail data to segment customers and discover product purchasing patterns. It provides actionable data-driven recommendations for targeted marketing and product bundling.

## Methodologies
* Data Preprocessing and Scaling
* Principal Component Analysis (PCA)
* Customer Segmentation: K-Means and DBSCAN Clustering
* Market Basket Analysis: Apriori Algorithm

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, MLxtend, Matplotlib, Seaborn

## Key Findings
* Identified distinct customer segments based on purchasing behavior and recency.
* Discovered strong association rules between specific products to optimize cross-selling.
* Evaluated cluster quality and separation using silhouette scores.
